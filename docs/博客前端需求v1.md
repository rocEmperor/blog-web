一、注册页面
- 昵称：必填，4-20 位，字母开头，字母 / 数字 / 下划线
- 邮箱：必填，邮箱符合邮箱格式
- 密码：必填，8-20 位，字母 + 数字
- 操作流程：输入信息 → 校验通过 → 注册成功 → 自动跳转至登录页面；
- 交互：各种校验通过后方可提交，密码和确认密码需要相同，提交增加loading，防止快速点击重复提交；
二、登录页面
- 昵称：必填；
- 密码： 必填；
- 操作流程：登录成功 → 自动跳转至博客首页；
- 交互：各种校验通过后方可提交，提交增加loading，防止快速点击重复提交；
三、账号退出
- 操作流程：退出成功 → 自动跳转至博客首页；
- 交互：提交增加loading，防止快速点击重复提交；
四、忘记密码页面
- 邮箱：必填，邮箱符合邮箱格式
- 验证码： 必填
- 密码：必填，8-20 位，字母 + 数字
- 确认密码：必填，与密码必须相同
- 操作流程：重置密码成功 → 自动跳转至登录页面；
- 交互：密码和确认密码需要相同，提交增加loading，防止快速点击重复提交；
五、账号设置-资料与头像设置页面
- 头像：非必填，支持png/jpg/jpeg格式
- 昵称： 必填，4-20 位，字母开头，字母 / 数字 / 下划线
- 个人简介：必填，8-50 位
- 手机号：非必填，如果有符合手机号格式
- 交互：提交增加loading，防止快速点击重复提交，设置成功提示设置成功；
- 交互：上传头像功能：上传给后端文件流，后端返回新的图片的url
六、账号设置-资料与头像设置页面
- 邮箱：必填，邮箱符合邮箱格式
- 当前密码： 必填，4-20 位，字母开头，字母 / 数字 / 下划线
- 新密码：必填，4-20 位，字母开头，字母 / 数字 / 下划线
- 确认新密码： 必填，与新密码必须相同
- 交互：提交增加loading，防止快速点击重复提交，设置成功提示设置成功；
七、账号设置-注销账号页面
- 操作流程：注销成功 → 自动跳转至登录页面；
- 交互：点击二次提示：是否注销该账号，提交增加loading，防止快速点击重复提交，设置成功提示注销成功；
八、 新建博客文章页面
- 标题：必填，4-30字
- 可见范围：必填，枚举值：公开：open；仅自己可见：only_myself
- 文章分类：必填，枚举值：后端：‘1’；前端：‘2’；Android：‘3’；iOS：‘4’；人工智能：‘5’；工具开发：‘6’；阅读：‘7’
- 正文：必填，富文本字符串，最多能存5000字
- 交互：校验所有字段合法后可提交，提交增加loading，防止快速点击重复提交，新建成功后自动跳转我的博客列表页
九、 编辑博客文章
- 标题：必填，4-30字
- 可见范围：必填，枚举值：公开：open；仅自己可见：only_myself
- 文章分类：必填，枚举值：后端：‘1’；前端：‘2’；Android：‘3’；iOS：‘4’；人工智能：‘5’；工具开发：‘6’；阅读：‘7’
- 正文：必填，富文本字符串，最多能存5000字
- 交互：校验所有字段合法后可提交，提交增加loading，防止快速点击重复提交，新建成功后自动跳转我的博客列表页
十、 查看我的博客文章列表
- 包含字段：标题，状态，更新时间
- 交互：支持分页，操作列有编辑操作：点击跳转编辑页面，携带博客ID；删除操作：点击有二次确认弹窗，提交增加loading，防止快速点击重复提交，删除成功后自动刷新列表接口；标题列支持超链接，点击标题可跳转至博客详情页面；支持按博客名称模糊查询，回车键入后触发查询
十一 查看文章详情页面
- 文章包含字段：标题，更新时间，博客正文，博客分类，点赞数量
- 当前文章的评论列表包含字段：评论人昵称，评论人头像，评论更新时间，评论内容
- 发表评论提交：提交内容字段：必填，1-300字
- 交互：发表评论提交增加loading，防止快速点击重复提交。提交成功后刷新评论列表
十二 公开博客列表页面
- 列表包含字段：标题，用户昵称，博客类型，更新时间，点赞数量，评论数量，博客中文
- 交互：列表支持分页，下拉翻页刷新，左侧菜单支持通过分类查询博客列表；支持通过博客标题模糊查询；点击列表点赞和评论按钮（需要判断登录状态，如何登录则跳转详情页，如果未登录则跳转登录页，登录成功后返回当前操作页）
十三 个人信息页面
- 展示个人信息：昵称，头像，个人简介
- 公开文章列表：展示更新时间，博客类型，标题，博客正文
- 交互：列表支持分页，分页，默认每页十条，下拉翻页刷新
十四 头部导航
- 交互：未登录状态下：右侧操作展示登录/注册/用户默认头像；用户登录状态下：右侧操作栏展示我的博客/账号设置/退出/用户名
